# Infotech Wizard Assignment

A responsive landing page built using HTML, CSS, and JavaScript based on the provided Figma design reference.

## Features

- Fully responsive design
- Clean and modern UI
- Responsive navigation menu
- Hero section with call-to-action buttons
- Compliance information section
- Interactive hover effects
- Mobile and tablet friendly layout

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/DMANOJKUMAR16/infotech-wizard-assignment
```

2. Open the project folder

```bash
cd infotech-wizard-assignment
```

3. Run the project

Open `index.html` in your browser.
